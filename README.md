# 📊 Short Video Consumption Impact Analysis (SQL Project)

## 📌 Project Overview

This project analyzes the impact of short video consumption (Reels, Shorts, TikTok) on attention span, focus levels, and productivity using SQL.

With the increasing use of short-form video platforms, this analysis explores how screen time behavior affects cognitive performance and daily task efficiency.

---

## 🎯 Objectives

* Analyze the relationship between short video usage and attention span
* Evaluate the impact of screen time on focus levels
* Study how sleep affects productivity
* Identify stress patterns based on digital usage
* Compare the impact across different platforms

---

## 🛠️ Tools & Technologies

* MySQL
* SQL (Aggregation, Group By, Case Statements)

---

## 📂 Dataset Description

The dataset contains user behavioral data including:

* Age
* Reels watch time (hours)
* Daily screen time (hours)
* Sleep hours
* Attention span score (1–10)
* Focus level (1–10)
* Task completion rate (%)
* Stress level (Low, Medium, High)
* Platform used (Instagram, YouTube, TikTok)

---

## 🔍 Analysis Performed

### 1. Overall Summary

* Average reels usage, attention, focus, and productivity

### 2. Reels Usage vs Attention

* Analyzed how increasing reels consumption affects attention span

### 3. Screen Time vs Focus

* Studied the impact of total screen time on user focus

### 4. Sleep vs Productivity

* Evaluated how sleep duration influences task completion

### 5. Stress Analysis

* Compared stress levels with screen time and attention

### 6. Platform Comparison

* Compared attention, focus, and productivity across platforms

### 7. Usage Category Analysis

* Grouped users into Low, Medium, and High usage categories

### 8. High Performer Analysis

* Identified behavioral patterns of highly focused users

---

## 📊 Key Insights

* High short-video consumption is associated with lower attention span
* Increased screen time negatively impacts focus levels
* Users with adequate sleep show higher productivity
* Higher stress levels are linked with increased screen usage
* Platform usage shows variation in cognitive impact
* Low usage users demonstrate better focus and attention
